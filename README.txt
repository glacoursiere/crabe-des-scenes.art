CRABE DES SCÈNES — LIVRAISON RC33
=================================

CONTENU DU SITE
---------------
- Six pages : Accueil, La Troupe, Spectacles, Projet, Ateliers et Contact.
- Feuille de style dans css/style.css.
- JavaScript dans js/script.js.
- Images, affiches, logos et partenaires dans le dossier images.
- Réservation de Piratages et inscription aux ateliers.
- Panneau de sociofinancement vers La Ruche.

FICHIERS TECHNIQUES
-------------------
- favicon.ico et icônes PNG pour navigateurs, Apple et Windows.
- site.webmanifest pour l'identité installable du site.
- sitemap.xml pour aider les moteurs de recherche à découvrir les pages.
- robots.txt pour indiquer l'adresse du plan du site.
- browserconfig.xml pour les anciennes tuiles Windows.
- CNAME pour conserver crabe-des-scenes.art sur GitHub Pages.
- 404.html pour afficher une page d'erreur cohérente avec le site.
- images/social-preview.jpg pour les aperçus sur les réseaux sociaux.
- Métadonnées canoniques, Open Graph et Twitter dans chaque page.
- Données structurées Schema.org sur la page d'accueil.
- Numéro de version RC33 ajouté au CSS et au JavaScript pour contourner le cache.
- Typographie American Typewriter avec Courier Prime comme solution Web multiplateforme.
- Nouvelle affiche piratage_represntations_v2.jpg sur Accueil et Spectacles.
- Accueil : titre visuel retiré, logo et mention de la troupe agrandis.
- Titres Ateliers de jeu et Écrivez-nous adaptés sur une seule ligne.
- Spectacles : nouvelle affiche affichée intégralement, sans recadrage.
